### Технологии и инструменты.

#### Языки и фреймворки

- Python - для работы с ИИ.
- Golang - для финтехнологий.
- Java - для финтехнологий, для АПИ портала самообслуживания.
- SpringBoot - для АПИ портала самообслуживания.
- React - для портала самообслуживания.

#### Инструменты

- Airflow - для доставки данных в отчёты
- Docker - для независимости контейнеров друг от друга и простоты управления процессами развертывания.
- CI/CD - для автоматизации выпуска релизов и развёртывания.
- Power Builder - клиентское ПО для работы с DWH.
- Power BI - для аналитики.
- Apache Camel - шина интеграции.

#### Платформы

- Microsoft SQL Server 2008 - для DWH.
- DWH
- Postgresql - БД для хранения шаблонов отчётов портала самообслуживания.
- ClickHouse - для хранения отчётности.
- Облачные сервисы (Yandex Cloud) - для масштабирования отчетности.
